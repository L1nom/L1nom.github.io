The first project I spent time on was to classify handwriting input from a user. When searching up beginner data science projects, this one interested me for its simplicity and complexity. League of Legends Data Analysis

I wanted to try analysing data on something I take see regularly, and what better way than to use data from the game I have played for 6 years! Using the RIOT Games API, I was able to extract data related to an account, such as wins-loss or match history logs. Along with the use Streamlit, I was able to display the data fetched from a give account, and then show related information on a web interface
